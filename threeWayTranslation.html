Three-Way Translation

Author: Philip Dorrell

Correspondence is a framework for interactively visualizing correspondences between items in related structures.

When comparing two structures, granularity of corresponding items can be chosen based on the relationships between the two structures.

With three (or more) structures, there is no clear best choice of granularity, because for each structure, there are multiple relationships competing to determine the granularity.

A solution to this problem is to choose item numbers based on the finest granularity, and then use multiple IDs for items that refer to multiple items in other structures.

The mouse-over interaction is then augmented to include partial highlighting. An item will be partially highlighted whenever its IDs overlap with those of the selected item, but the partially hightlighted item has at least one ID which is not an ID of the selected item.

Applying this to the concepts of "siblings" and "cousins", for items that have the same ID as a selected item in the same structure or different structure respectively, this results in a total of six different types of highlighting:

  1. The Selected Item
  2. A "sibling" in the same structure that shares all its IDs with the selected item
  3. A "partial sibling" in the same structure that shares some of its IDs with the selected item
  4. A "cousin" in a different structure that shares all its IDs with the selected item
  5. A "partial cousin" in a different structure that shares some of its IDs with the selected item
  6. An unrelated item that shares no IDs with the selected item.

So, for example, if the selected item had IDs of 1,2,3, then the following IDs for items in a different structure would give these matches:

  • 1: "cousin"
  • 1,2: "cousin"
  • 1,2,3: "cousin"
  • 1,2,3,4: "partial cousin"
  • 1,3,4: "partial cousin"
  • 4: "unrelated"

In practice both partial siblings and partial cousins can be displayed using a common "partial" highlight, so that only five different colorations are required.
